LogMonitor 3.6

Горячие клавиши

    Поиск

  • <Ctrl + F> - Поиск текста
  • <F3> - Поиск следющего вхождения искомого текста
  • <Shift + F3> - Поиск предыдущего вхождения искомого текста

    • Фильтрация

  • <Ctrl + G> - Показывать строчки, содержащие указанный текст (фильтр показа)
  • <Ctrl + Shift + G> - Показывать строчки, НЕ - содержащие указанный текст (фильтр скрытия)

    • Блочная фильтрация

    (активна, если задан blockPattern для лога)
  • <Ctrl + B> - Показывать блоки лога, содержащие указанный текст (блочный фильтр показа)
  • <Ctrl Shift + B> - Показывать блоки лога, НЕ - содержащие указанный текст (блочный фильтр скрытия)

    • Прочее

  • <F1> - Текущее окно
  • <ESC> - Скрыть окно лога
  • <С> - Очистить окно лога
  • <F5> - Перезагрузить данные лога
  • <Shift + F5> - Сбросить фильтры и перезагрузить лог
  • <W> - Вкл./выкл. перенос строк
  • <N> - Вкл./выкл. нумерацию строк
  • <Ctrl + S> - Поиск похожих блоков

Использование мыши

  • Двойной клик - показ в сплывающем окне блока лога с форматирование входящего в него XML
  • Выделение - остановка автоскролла, выделенный текст автометически копируется в буффер обмена
